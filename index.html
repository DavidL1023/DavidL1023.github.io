    Project Introduction

    Ahead are a select few projects that I've worked on, I made these with the goal in mind of learning new aspects of design with each one. While I do want to specialize in C++, I also wanted to try out many tastes of programming in order to broaden my skill set.

    This website

    This website was designed and programmed entirely by me, it features a parallax background, animated movement, responsive screen sizes, and some plugin use.

    Regimen: Build habits

    I created this app in Kotlin using Google's Jetpack Compose environment, it offers a clean and animated UI and a functional way to write notes and keep reminders.

    Bully bot

    This bot was developed in version 13 of Discord.js, when invited to a server it allows for you to set a target that will be made fun of. It was developed in order to work on multiple servers in parallel.

    A project I created by request, this locally ran website offers a fast way to see all of your favorites between multiple art/anime imageboards. It uses REST api, coroutines, threads, and has its own backend.

    A robotics project where I used an arduino to build a car that reacts to the environment using ultrasonic sensors. It also has a manual driving mode using a controller.
